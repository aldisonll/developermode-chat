# developermode-chat
A websocket chat made with socket.io and express.io
## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install socket and express.

```bash
npm i socket.io socket.io
```
## Usage
```bash
cd server
node socket-chat.js
```
